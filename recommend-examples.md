# 문구 추천 예시

"문구 추천받기"가 사용하는 예시 모음입니다. **이 파일을 수정한 뒤 터미널에서 `npm run build`를 실행하고, Figma에서 플러그인을 다시 실행하면 반영됩니다.**

## 작성 방법

- 예시 하나는 **`### 원본`** 한 줄과, 그 아래 **`- 추천안`** 여러 개로 이뤄집니다.
- 추천안 안에서 **줄을 바꾸고 싶으면 ` / ` (앞뒤 공백 포함 슬래시)** 로 표시하세요. 플러그인에서 두 줄로 보여집니다.
- 사용자가 입력한 문구가 `원본`과 (공백·문장부호 무시하고) 같거나, 서로 포함하면 그 추천안들을 보여줍니다.
- 매칭할 때 **마스킹된 이름(홍\*동), 숫자(전화번호·"외 2명" 등)는 무시**합니다 — 이름·수량·번호만 다른 문구도 같은 예시로 잡혀요. 단, 추천안에 적어둔 이름·숫자는 그대로 나오니 실제 값에 맞게 고쳐 쓰세요.
- 제목(`##`)과 `###`, `-` 기호는 형식이니 바꾸지 마세요.

## 스타일 원칙 (참고 — 자세한 내용은 ux-writing.md 가이드)

- 해요체, 부드러운 종결(`~있어요` `~돼요` `~없어요` `~해 주세요`)
- 2단 구조: **첫 줄=상황 설명 → 둘째 줄=다음 행동**(결정은 `~할까요?`, 행동 유도는 `~해 주세요`)
- 능동적 말하기(됐어요→했어요), 긍정적 말하기(없어요→~하면 할 수 있어요)
- 캐주얼한 경어(~시겠어요?→~할까요?), 명사+명사 풀어쓰기(잔액 부족으로→잔액이 부족해서)
- 간결하고 쉬운 말 (전송→보내다), 부정 상황도 딱딱하지 않게("찾기 실패"❌ "찾을 수 없어요"✅)

## 추천 예시

### 진행하던 작업이 있습니다. 계속하시겠습니까?
- 진행 중인 내역이 있어요. / 이어서 진행할까요?

### 공유 요청을 취소하면 요청 내역이 삭제됩니다. 취소하시겠습니까?
- 취소할 경우 요청 내역도 삭제돼요. / 공유 요청을 취소할까요?

### 기기를 찾지 못했습니다. QR코드를 다시 스캔하세요.
- 기기를 찾을 수 없어요. / QR코드를 다시 스캔해 주세요.

### 보호자가 허락하기 전에는 가입할 수 없어요
- 보호자가 허락해야 가입할 수 있어요

### 지금 버전에서는 쓸 수 없어요. 생체 인증을 쓰려면 앱을 최신 버전으로 업데이트 해주세요.
- 앱을 업데이트해주세요. / 생체 인증을 쓰려면 최신 버전이 필요해요.

### 어떤 목적으로 대출받으시나요?
- 대출 목적이 무엇인가요?

### 어떤 이유로 신고하시나요?
- 신고 이유를 선택해 주세요.

### 잔액 부족으로 구매하지 못했어요
- 잔액이 부족해서 구매하지 못했어요

### 홍*동(010-1234-5678) 외 2명에게 권한 삭제 알림톡을 전송할까요?
- 권한 삭제 알림톡을 보내려고 해요. / 홍*동(010-1234-5678) 님 외 2명에게 보낼까요?
- 홍*동(010-1234-5678) 님 외 2명에게 권한 삭제 알림톡을 보낼까요?
- 권한 삭제 알림톡을 홍*동(010-1234-5678) 님 외 2명에게 보낼까요?
